# HR-Agent

AI-powered HR support agent built using Microsoft Copilot Studio to automate employee HR queries and improve self-service support.

# Overview

This project demonstrates how AI agents can streamline HR operations by answering common employee queries such as:

* Attendance information
* Salary-related queries
* Leave policies
* Paternity leave information
* Office timings
* Attendance regularization
* Handling non-HR related queries

The agent was developed using Microsoft Copilot Studio and follows Agent Academy concepts for declarative agents and enterprise AI assistance.

# Setup Instructions

## Prerequisites

* Microsoft Copilot Studio access
* Microsoft 365 account
* Required licensing enabled

## Steps

1. Open Microsoft Copilot Studio

2. Create a new Copilot agent

3. Configure the agent instructions.

4. Upload or connect the required HR knowledge sources

5. Add and configure HR query scenarios

6. Publish the agent

7. Test the HR scenarios using the demo cases

# Architecture

The overall solution architecture is available in:

* `Architecture.pdf`
 ## Architecture

The HR-Agent architecture is built using Microsoft Copilot Studio and Microsoft 365 Copilot concepts. The user interacts with the agent through a conversational interface, where queries are processed by the declarative agent. The orchestration layer routes requests and retrieves relevant HR information from the SharePoint Knowledge Base. The agent then generates context-aware responses for employee HR queries such as attendance, leave, salary, and office policies. This architecture demonstrates enterprise AI automation using Microsoft Copilot technologies.
